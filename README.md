## Projects

### [Remote MCP Server](https://github.com/KhizarA77/typescript-sdk)
Production-grade remote MCP server in TypeScript using **Streamable HTTP transport** with Bearer token authentication. Deployed behind **nginx** as a reverse proxy with **systemd** process management, targeting Claude desktop/API integration. Built on and contributed back to the official [`modelcontextprotocol/typescript-sdk`](https://github.com/modelcontextprotocol/typescript-sdk).

`TypeScript` `MCP` `Nginx` `systemd` `Bearer Auth` `Claude`

---

### [E-Commerce Big Data Pipeline](https://github.com/KhizarA77/ecommerce-big-data-pipeline)
End-to-end distributed data pipeline processing real-time e-commerce transactions. **Kafka** event streaming across 12 topics (fraud detection, demographic analytics, payment insights) → **Apache Spark** consumers → **HBase** storage. Node.js producer, Python analytics, Docker Compose orchestration, Airflow DAGs, and a live fraud-detection ML model.

`Python` `Node.js` `Kafka` `Apache Spark` `HBase` `Airflow` `Docker`

---

### [GitHub Repository Crawler](https://github.com/KhizarA77/github-crawler)
Scalable crawler that indexes **100,000+ public GitHub repositories** via the GraphQL API into PostgreSQL. Clean architecture with repository/service separation, rate-limit retry logic, upsert-safe schema, and GitHub Actions for daily automated runs. Designed for horizontal scale: sharded workers, async I/O, streaming inserts.

`Python` `PostgreSQL` `GitHub GraphQL API` `GitHub Actions`

---

### [Azure Blob Storage Manager](https://github.com/KhizarA77/cc_blob)
Express.js service with secure file lifecycle management over Azure Blob Storage — upload, list, and delete blobs via a web UI. Generates scoped **SAS URLs** with 1-hour TTL for read-only access, UUID-namespaced blob keys, and Azure deployment via GitHub Actions.

`Node.js` `Express` `Azure Blob Storage` `EJS` `CI/CD`
